ResourceManager Class

Defined in: ../Source/Runtime/Resources/ResourceManager.js:3
Module: Resources

Resource manager is used to manage available resources used by objects The resource manager is used to extend the Program object, and it's not designed to be used as a standalone The manager is used to manage the following types of resources

  • Images
  • Videos
  • Audio
  • Fonts
  • Textures
  • Materials
  • Geometries
