KinectDevice Class

Extends THREE.Object3D
Defined in: ../Source/Runtime/Objects/Device/KinectDevice.js:3
Module: Devices

Kinect device object This object is used to connect Gorlot to a Microsoft Kinect v1, it only works in Microsoft Windows. The operation of the kinect object depends on a server program used to connect to kinect that sends the data to nunuStudio via WebSocket.
